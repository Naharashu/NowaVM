# NowaVM

## Компіляція

Ви можете компілювати .asm файли, що мають ВМ асамблерний код у .bin.

Наприклад:
```
ld r0 42
hlt
```

буде скомпільовано у out.bin:
```
0x01 0x00 0x2A 0x00 0x00 0x00 0x00 0x00 0x00 0x00 0xFF
```

але також воно згенерує out.sym, що містить інформацію для nwld.

наприклад b.asm

```
func:
ld r0 42
ret

funcx:
hlt
```

b.sym буде:

```
EXPORT funcx 11
EXPORT func 0
```


## Стандартна бібліотека

### entry0.asm

entry0.asm це файл, який має тільки один рядок коду:

```
jmp _start
```

Його головне застосування як стандартна точко входу.

## VM прапорці

--help - виводить допоміжне повідомлення

-o - встановлює назву файлу бінарника

-interpret - використовує інтерпретатор замість JIT(по замовчуванню на aarch64)

-r - виводить вміст регістрів

-V - verbose режим(виводить дебаг інформацію)

-v - виводить версію

-e - виконує байткод(-e file.name якщо не компілюєте)

-Wrt - виводить рантайм помилки

-fno-entry0 - не включає entry0.asm

-opt - вмикає оптимізації
